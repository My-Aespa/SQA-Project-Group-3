กําหนดให้ใช้ Factory Method Design Pattern ในการเขียนโค้ดตาม Requirement ที่กําหนดให้โดยเขียนโค้ดด้วยภาษา Java
Requirement ร้านกาแฟ: ต้องการสร้างโปรแกรมรับออเดอร์สำหรับร้านคาเฟ่ โดยมีเมนูต่างๆดังนี้ 
1. เครื่องดื่ม (Beverages) 
1.1กาแฟ (Coffee) -เอสเปรสโซ (Espresso) -คาปูชิโน (Cappuccino) -ลาเต้ (Latte) -มอคค่า (Mocha) -อเมริกาโน (Americano) 
1.2 ชา (Tea) -ชาดำ (Black Tea) -ชาเขียว (Green Tea) -ชาอู่หลง (Oolong Tea) -ชามะนาว (Lemon Tea) 
1.3 เครื่องดื่มเย็น (Cold Beverages) -สมูทตี้ผลไม้ (Fruit Smoothies) -นมปั่น (Milkshakes) -น้ำผลไม้ (Fruit Juices) 
2. ขนม (Pastries) 
2.1 ขนมปัง (Bread) -ครัวซองต์ (Croissant) -บัน (Bun) -ขนมปังสังขยา (Custard Bread) 
2.2 เค้ก (Cakes) -เค้กช็อกโกแลต (Chocolate Cake) -เค้กสตรอว์เบอร์รี่ (Strawberry Cake) -เค้กมะพร้าว (Coconut Cake) 
2.3 คุกกี้ (Cookies) -คุกกี้ช็อกโกแลตชิป (Chocolate Chip Cookies) -คุกกี้โอ๊ต (Oat Cookies) 
3. อาหารว่าง (Snacks) 
3.1 แซนวิช (Sandwiches) -แซนวิชไก่ (Chicken Sandwich) -แซนวิชทูน่า (Tuna Sandwich) 3.2 พาสต้า (Pasta) -สปาเก็ตตี้ (Spaghetti) -เพนเน่ (Penne) 
3.3 สลัด (Salads) -สลัดผัก (Vegetable Salad) -สลัดไก่ (Chicken Salad) 
เมื่อลูกค้าสั่งรายการต่างๆเข้ามา ทางร้านจะทำตามรายการนั้น 
#### 
จากโค้ดที่สร้างเขียน JUnit เพื่อทดสอบโค้ดที่กําหนดให้ โดย Statement coverage เป็น 100% และ Branch coverage เป็น 100%
